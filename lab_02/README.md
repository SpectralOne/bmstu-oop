При сборке и тестировании используется библиотека гугол тестов (Google Test) и gcc версии 9+.
Без них, соотвественно, не соберётся, не запустится...

Сборка:

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
Запуск Тестов:

```bash
ctest -V -R Testing
```
